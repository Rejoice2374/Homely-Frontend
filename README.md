# 🏡 Homely Frontend

A modern frontend application for the **Homely platform**, designed to provide users with a seamless experience for browsing and managing properties.

---

## 🚀 Overview

Homely Frontend is the client-side application that interacts with the Homely API. It allows users to:

- View property listings
- Create and manage properties
- Authenticate (login/register)
- Interact with a clean and responsive UI

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript
- Material UI
- Fetch API
- REST API integration
